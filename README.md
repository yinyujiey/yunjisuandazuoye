# 2025年秋云计算期末大作业 - 智能体云原生开发

## 1. 项目选题
**命题二：学习效果评估和巩固智能体**

## 2. 项目简介
本项目开发了一个基于云原生架构的智能学习助手。针对学完新知识后缺乏客观评估手段的痛点，系统实现了从 PDF 资料上传、知识点提取、自动出题（选择题+简答题）、智能判卷到“错题本”自动收集的全闭环学习评估系统。

## 3. 核心功能
- **语义解析**: 利用 `PyMuPDF` 自动识别并提取 PDF 文档中的分层结构与核心知识点。
- **动态出题**: 基于 DeepSeek-V3 大模型，针对资料内容自动生成 2 道选择题和 1 道简答题，确保题目覆盖核心考点。
- **智能判卷**: AI 模拟阅卷老师，对用户的简答题进行语义对比评分，并给出详尽的个性化纠偏建议。
- **弱点记忆**: 自动识别用户高频错误及低分题目，利用 **Redis** 列表实现“错题本”的自动持久化收集。

## 4. 技术架构与云原生特征
项目采用微服务化设计，完全符合“架构优先”的云原生特征：
- **后端框架**: FastAPI (高性能异步 Web 框架)
- **智能体引擎**: LangChain + DeepSeek-V3 API
- **状态存储**: Redis (实现用户学习状态的持久化记忆)
- **容器化部署**: 编写 `Dockerfile` 并使用 `Docker Compose` 进行多容器编排，实现一键部署环境。
- **容错设计**: 在接口层增加了 **Check Layer**，解决了大模型生成 JSON 字段不一致（幻觉）的稳定性问题。
